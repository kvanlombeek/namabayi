Wat is Namabayi

Namabayi, Indonesisch voor babynaam, helpt je bij kiezen van een naam.

Je kan statiestieken opzoeken over een naam dat je in gedachten hebt. Of je kan de applicatie zelf suggesties laten doen. Hoe meer suggesties je liked, hoe intelligenter het suggestie mechanisme wordt.

De statistieken bij de namen zijn zelf berekend en als volgt bepaald:

  • Origineel: Is de naam vaak gegeven?
  • Populair: Is de naam vaak gegeven de laatste vijf jaar?
  • Klassiek: Is de naam populair over alle leeftijden heen?
  • Vintage: Was de naam populair bij de 65 plussers en erna niet meer?
  • Trending: Is de populariteit van de naam de laatste jaren aan het stijgen?

Veel succes bij de leuke zoektocht naar een naam!

About

Altough there are many services available for picking a baby name, on a long holiday I decided to build our own. I had in mind that it should be possible to calculate features for each name (time series features, text analysis features and spatial features) and build a recommendation engine on top of them.

This project demonstrates what I like best, taking an analysis (defining the features and building the model) a step further and build an interactive visualisation around it. By doing so, you can test it out with many examples, using it where ever in the world and send it around to other people.
